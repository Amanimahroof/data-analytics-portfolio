# Data Analytics Portfolio

## Overview
This repository contains a collection of data analytics projects developed to demonstrate practical skills in data analysis, visualization, SQL, and dashboard development. Each project focuses on analyzing real-world datasets and extracting meaningful insights using industry-relevant tools.

The portfolio highlights hands-on experience with Python, R, SQL, and Power BI through structured analysis, KPI-driven insights, and clear communication of findings for business decision-making.

---

## Projects

### 1. Loan Approval Analysis (R)
**Tools:** R, RStudio  
**Focus:** Exploratory Data Analysis  

This project analyzes a loan approval dataset to understand the key factors influencing loan approval decisions. The analysis includes data exploration, preprocessing, and statistical summaries to identify patterns related to applicant profiles and loan outcomes.

**Key Highlights:**
- Data cleaning and preprocessing in R  
- Exploratory analysis of applicant attributes  
- Identification of patterns influencing loan approval decisions  

**Folder:** Loan-Approval-Analysis-R



### 2. Supermarket Sales Analysis (Python)
**Tools:** Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn  
**Focus:** Exploratory Data Analysis and Visualization  

This project performs exploratory data analysis on a supermarket sales dataset to examine sales trends, customer behavior, and product performance using Python-based analytics techniques.

**Key Highlights:**
- Dataset inspection and validation  
- Descriptive statistical analysis  
- Sales and customer behavior analysis  
- Data visualization for insight generation  

**Folder:** Sales-Dashboard-Analysis-Python


### 3. Supermarket Sales Analysis (SQL)
**Tools:** MySQL, SQL, Jupyter Notebook  
**Focus:** Database Design and Analytical Querying  

This project demonstrates relational database design and SQL-based analysis using supermarket sales data. The dataset is structured into normalized tables, and SQL queries are used to extract meaningful business insights.

**Key Highlights:**
- Database creation and table design  
- Use of primary and foreign keys  
- Data normalization and insertion  
- Analytical SQL queries for business insights  

**Folder:** supermarket-sale-SQL-Analysis



### 4. Delhi Air Quality Index (AQI) Analysis (Power BI)
**Tools:** Power BI  
**Focus:** Data Visualization and Dashboarding  

This project analyzes Delhiâ€™s Air Quality Index (AQI) data from 2020 to 2024 using Power BI. The interactive dashboard highlights pollution severity, seasonal patterns, and the influence of PM2.5 on air quality levels.

**Key Highlights:**
- KPI cards for AQI and PM2.5 levels  
- Time-based trend analysis  
- Seasonal pollution pattern visualization  
- Insight-driven dashboard design  

**Folder:** Delhi-AQI-PowerBI-Analysis



### 5. HR Attrition Analysis (Business Analytics)
**Tools:** Python, Power BI  
**Focus:** Business Analytics and Dashboarding  

This project analyzes employee attrition data to identify key factors contributing to workforce turnover and to support data-driven HR decision-making. The analysis focuses on understanding attrition patterns across departments, income levels, tenure, workload, and employee experience metrics.

**Key Highlights:**
- Business-focused feature selection and data preparation  
- Attrition rate analysis across departments and employee segments  
- Evaluation of attrition drivers such as income, overtime, tenure, job satisfaction, and work-life balance  
- Interactive Power BI dashboard with KPIs and filters for HR insights  

**Folder:** HR-Attrition-Analysis-BA
