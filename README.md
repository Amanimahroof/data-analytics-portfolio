# Data Analytics Portfolio

## Overview
This repository contains a collection of data analytics projects developed to demonstrate practical skills in data analysis, visualization, SQL, and dashboard development. Each project focuses on analyzing real-world datasets and extracting meaningful insights using industry-relevant tools.

The portfolio highlights hands-on experience with Python, R, SQL, and Power BI through structured analysis and clear insight communication.


## Projects

### 1. Loan Approval Analysis (R)
**Tools:** R, RStudio  
**Focus:** Exploratory Data Analysis  

This project analyzes a loan approval dataset to understand key factors influencing loan approval decisions. The analysis includes data exploration, data preprocessing, and statistical summaries to identify patterns related to applicant profiles and loan outcomes.

**Key Highlights:**
- Data cleaning and preprocessing in R  
- Exploratory analysis of applicant attributes  
- Identification of patterns influencing loan approval  

Folder: Loan-Approval-Analysis-R



### 2. Supermarket Sales Analysis (Python)
**Tools:** Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn  
**Focus:** Exploratory Data Analysis and Visualization  

This project performs exploratory data analysis on a supermarket sales dataset to examine sales trends, customer behavior, and product performance using Python-based data analysis techniques.

**Key Highlights:**
- Dataset inspection and validation  
- Descriptive statistical analysis  
- Sales and customer behavior analysis  
- Data visualization for insight generation  

Folder: Sales-Dashboard-Analysis



### 3. Supermarket Sales Analysis (SQL)
**Tools:** MySQL, SQL, Jupyter Notebook  
**Focus:** Database Design and Analytical Querying  

This project demonstrates relational database design and SQL-based analysis using supermarket sales data. The dataset is structured into normalized tables, and SQL queries are used to extract business insights.

**Key Highlights:**
- Database creation and table design  
- Use of primary and foreign keys  
- Data insertion and normalization  
- Analytical SQL queries for business insights  

Folder: supermarket-sale-SQL-Analysis
