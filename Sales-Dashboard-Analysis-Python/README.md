# Supermarket Sales Analysis (Python)

## Project Overview
This project showcases Python-based exploratory data analysis performed on a supermarket sales dataset. The objective is to analyze sales performance, customer behavior, and product trends using data analysis and visualization techniques.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Loaded and explored the supermarket sales dataset
- Checked data structure and consistency
- Performed descriptive statistical analysis
- Analyzed sales by product line and branch
- Studied customer purchasing patterns
- Visualized key insights using charts and plots

## Files in this Folder
- supermarket_sales_analysis.py.ipynb – Python analysis notebook
- supermarket_sales.csv – Dataset used for analysis

This project is part of my Data Analytics portfolio and demonstrates my Python data analysis and visualization skills.
